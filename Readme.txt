System Requirements:
Have python version 2.7 installed.

How to Run:
Method 1:
Open file entertainment_center.py in Idle and  press F5 key to run. 
You can also click on Run -> Run Module to run/ launch the web browser to display the movies page.

Method2:
Open the command prompt and navigate to the folder containing the project files.
Type "python entertainment_center.py" at the command prompt and press enter to run/ launch the web browser to display the movies page.
